Pagetemplate

Deze website is een template die door deelnemers naar eigen inzicht gebruikt kan worden voor eigen projecten: Externe link - HTML Websitetemplate Bee Team LKQ.

Het onderwerp dat voor deze template is gekozen is het project Bee Team LKQ in Heerlen.

Zoals de hele site een template is voor een website, zo is deze pagina 'template.html' een pagetemplate die laat zien hoe de content van een webpagina gestructureerd kan worden. Hier is de code. Deze template om een pagina te structureren is slechts een voorbeeld. De stijl ervan is bewust sober en zakelijk gehouden. De template voor deze pagina is ook weer gebruikt voor alle andere pagina's.
