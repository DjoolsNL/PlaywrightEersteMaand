Template voor een cv-portfolioproject

  • De site is gebouwd als de menukaart van een fastfood restaurant: Gefrituurd, Hamburgers, Pizza, Drank en Toetjes (in ons geval: Leerbedrijf, Educatief, Baanperspectief, Portfolio en Templates). Aan die kapstok is alles opgehangen. Deze indeling zorgt voor een eenvoudige maar doeltreffende structuur.
  • Je kunt met de werkende testcode en / of met de werkende website als voorbeeld een mooi portfolio project voor je cv bouwen
